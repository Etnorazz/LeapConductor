Detailed Description

The Frame class represents a set of hand and finger tracking data detected in a single frame.

The Leap detects hands, fingers and finger-like tools within the tracking area, reporting their positions and orientations in frames generated at the Leap frame rate.

Access Frame objects through an instance of a Leap Controller. Implement a Listener subclass to receive a callback event when a new Frame is available.
